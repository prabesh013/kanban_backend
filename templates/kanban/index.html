KANBAN BACKEND API

  1. Install the requirements: pip install -r requirements.txt
  2. Create Database and Create various credentials information(User,Password) to work with Postgresql
  3. Migrate the database:python manage.py migrate
  4. For admin purpose Create Super User: python manage.py createsuperuser

Usage:

/api/board ---> GET, POST

/api/board/{id} ---> GET, PUT, PATCH

/api/list ---> GET, POST

/api/list/{id} ---> GET, PUT, PATCH

/api/card ---> GET, POST

api/card/{id} ---> GET, PUT, PATCH
